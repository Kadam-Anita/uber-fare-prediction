# uber-fare-prediction
📖 Project Overview

This project predicts Uber fares using machine learning techniques. It follows a complete ML pipeline from data cleaning to model deployment.

⚙️ Features
Data cleaning and preprocessing
Outlier detection using IQR
Feature engineering (distance, time features, etc.)
Regression modeling (Linear Regression / OLS)
Model evaluation (R² score, MAE, RMSE)
Model saving using .pkl
Prediction on new data

🛠️ Technologies Used
Python
Pandas, NumPy
Scikit-learn
Statsmodels
Matplotlib / Seaborn

📊 Workflow
Data collection
Data cleaning
Exploratory Data Analysis (EDA)
Feature engineering
Model training
Model evaluation
Model saving & prediction

🎯 Goal

To build a reliable model that can estimate Uber fares based on trip details and demonstrate real-world machine learning pipeline implementation.
